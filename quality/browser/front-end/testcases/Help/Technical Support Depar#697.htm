Test Case Description for Technical Support Departments

Purpose:

This basic testcase is designed to verify that there are clear options for tech support and easily accessible by the user

Initial Conditions:

Netscape should be started and a printer should be installed on the machine you are using

Steps/Description:

1. Select the toplevel Help menu.

2. Select Netscape Technical Support from the drop down menu

3. Under Departments on the left hand side click Home

4. Under Departments on the left hand side click Help Directory

5.


Expected Results:

3. Verify you are taken to http://help.netscape.com/index.html

4. Verify that you are taken to http://help.netscape.com/help_dir.html and there is and alphabetical listing of help products and services

5. Verify that you are taken to the details for this support option (Currently, you are not taken to the correct anchor position), also verify that clicking the link below this option takes you to http://www.decisionone.com/D1WWW/netscape/
