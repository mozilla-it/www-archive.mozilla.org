Localize These Files

This section tells you exactly what files need to be localized. The following tables contain two columns:
  • The file to be localized column gives the specific file name. The path name given for each file is relative to the Windows Front End Localization Kit's starting point; e.g. \kitRoot\products\client\windows\platform\de (German in this example). If the file or resource name is a hypertext link, further information can be found by following that link.
  • The purpose column describes the basic purpose of this file.
