Localization Notes for BOOKMARKS.HTM

As part of the Unlimited Localization Program, you may choose to include the default set of bookmarks that came with this kit, or you may create your own bookmarks file. Assuming you include a bookmarks file, you should save it to kitRoot\products\client\windows\platform\locale\nav40\program\defaults\bookmark.htm.
